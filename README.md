# SuperStore Sales Performance Dashboard

## Project Overview
This project is an interactive Sales Performance Dashboard built using Power BI, 
analyzing the SuperStore Sales Dataset to uncover key business insights across 
regions, categories, and time periods.

## Objectives
- Import and clean raw sales dataset
- Analyze monthly and yearly sales trends
- Identify top selling products and low performing items
- Perform region wise and category wise sales comparison
- Create KPIs for business decision making

## Tools Used
- Power BI Desktop
- Power Query (Data Cleaning)
- DAX (Data Analysis Expressions)
- Microsoft Excel

## Dataset
- Source: SuperStore Sales Dataset
- Records: 3,000+ rows
- Period: 2019 - 2020

## Key KPIs
- Total Sales: $1.57M
- Total Profit: $175.29K
- Total Orders: 3,003
- Profit Margin: 11.20%
- Total Quantity Sold: 22,314

## Dashboard Features
- Monthly Sales Trend (Line Chart)
- Sales by Region (Bar Chart)
- Sales by Category (Donut Chart)
- Sales by Segment (Donut Chart)
- Top 5 Products by Sales (Bar Chart)
- Sales by Ship Mode (Column Chart)
- Interactive Region and Year Slicers

## Key Insights
- West region generated the highest sales ($0.52M)
- Office Supplies is the top category (41.12%)
- Consumer segment dominates at 48.08%
- Standard Class is the most used shipping mode
- Q4 consistently shows highest sales performance
